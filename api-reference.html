API Reference

Financial infrastructure for AI agents. Issue virtual cards, set spending limits, track expenses.

BASE URL https://api.bankofagents.dev
All endpoints require the x-api-key header. Contact nik@irlll.com to get your key.
POST /v1/agents

Register a new agent. Creates an agent identity that can be issued virtual cards and tracked for spending.

Request body

PARAMTYPEREQDESCRIPTION
name string required A unique name for the agent
description string optional Human-readable description of what this agent does

Example request

# Register an agent
curl -X POST https://api.bankofagents.dev/v1/agents \
  -H "x-api-key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"name": "research-agent", "description": "Autonomous web research agent"}'

Example response

{
  "id": "agt_01hx7kp2z4n8m3qj",
  "name": "research-agent",
  "description": "Autonomous web research agent",
  "status": "active",
  "created_at": "2025-03-21T19:42:00Z"
}
GET /v1/agents

List all registered agents for your account. Returns agents ordered by creation date, newest first.

Query parameters

PARAMTYPEREQDESCRIPTION
limit integer optional Max results to return. Default: 20, Max: 100
offset integer optional Pagination offset. Default: 0

Example request

curl https://api.bankofagents.dev/v1/agents \
  -H "x-api-key: YOUR_API_KEY"

Example response

{
  "agents": [
    {
      "id": "agt_01hx7kp2z4n8m3qj",
      "name": "research-agent",
      "status": "active",
      "cards_count": 2,
      "created_at": "2025-03-21T19:42:00Z"
    }
  ],
  "total": 1
}
POST /v1/agents/{agent_id}/cards

Issue a virtual Visa card for the given agent via Stripe Issuing. The card is active immediately and ready to charge. Amounts are in cents.

Path parameters

PARAMTYPEREQDESCRIPTION
agent_id string required ID of the agent to issue a card to

Request body

PARAMTYPEREQDESCRIPTION
spending_limit integer optional Spending cap in cents (e.g. 10000 = $100.00)
spending_limit_interval string optional One of: per_authorization, daily, weekly, monthly

Example request

# Issue a card with $100/day spending limit
curl -X POST https://api.bankofagents.dev/v1/agents/agt_01hx7kp2z4n8m3qj/cards \
  -H "x-api-key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"spending_limit": 10000, "spending_limit_interval": "daily"}'

Example response

{
  "id": "card_a7f3k9p2m1",
  "agent_id": "agt_01hx7kp2z4n8m3qj",
  "status": "active",
  "last4": "4242",
  "brand": "Visa",
  "spending_controls": {
    "spending_limit": 10000,
    "spending_limit_interval": "daily"
  },
  "stripe_card_id": "ic_1PqR7sKX2eZvKYlo3J8Dq2T4",
  "created_at": "2025-03-21T19:42:00Z"
}

DELETE /v1/agents/{agent_id}/cards/{card_id}

Cancel a virtual card. Cancellation is immediate and permanent — the card will be declined on all future transactions. Cannot be reversed.

Path parameters

PARAMTYPEREQDESCRIPTION
agent_id string required ID of the agent
card_id string required ID of the card to cancel

Example request

# Immediately cancel a card
curl -X DELETE \
  https://api.bankofagents.dev/v1/agents/agt_01hx7kp2z4n8m3qj/cards/card_a7f3k9p2m1 \
  -H "x-api-key: YOUR_API_KEY"

Example response

{
  "id": "card_a7f3k9p2m1",
  "status": "cancelled",
  "cancelled_at": "2025-03-21T21:00:00Z"
}
